Reflection

Reflection 1: Developing Logical Thinking Through Flowcharts

In Lab Exercise 1 and Assignment 1, I practiced creating flowcharts using pre-test and post-test loops. This exercise helped me develop a clearer understanding of how to visually represent the flow of a program. I learned how to break down problems into smaller steps and organize them logically, which is a crucial skill in programming. This practice has strengthened my problem-solving abilities and prepared me for writing structured code.

Reflection 2: Enhancing Control Structures in C++ Programming

Completing Lab Exercise 2 deepened my understanding of control structures such as if-else statements, looping constructs, and the switch statement. These exercises taught me how to implement decision-making and repetitive tasks efficiently in C++ programs. I now feel more confident in using these constructs to write dynamic and flexible code. This knowledge has laid a strong foundation for tackling more complex programming challenges.

Reflection 3: Understanding Functions for Modular Programming

Through Lab Exercise 3, I gained valuable experience in creating and using both built-in and self-defined functions. This exercise gave me a clearer understanding of how functions work and how they can simplify code by promoting reusability and modularity. I now appreciate the importance of breaking down a program into smaller, manageable functions, which makes debugging and maintenance much easier.

Reflection 4: Optimizing Code with 2D Arrays

Assignment 2 introduced me to coding with 2D arrays, which has significantly improved my ability to handle and manipulate structured data. I learned how to store and process data in a more organized and efficient way. This exercise was particularly insightful as it showed me how arrays can be used to solve real-world problems, such as managing data in tables or grids. It was a valuable step in becoming a more proficient programmer.
